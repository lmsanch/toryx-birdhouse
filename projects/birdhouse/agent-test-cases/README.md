# Agent Test Cases

Manual agent-behavior test cases for Birdhouse.

These are lightweight human-readable test prompts you can run against a Birdhouse environment to validate multi-agent behavior and tool ergonomics.

Before running them, read `run-agent-test-suite.md`.

## Included Cases

- `run-agent-test-suite.md` - instructions for one agent to run the whole suite and report results
- `fib-recursive-agents.md` - recursive agent creation tree
- `read-exchange-tooling.md` - multi-turn conversation that exercises read modes and tool-call drill-down
- `implementation-tree-reading.md` - real multi-agent implementation tree with warm-up, coordination, handoffs, and checks
