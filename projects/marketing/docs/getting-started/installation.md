# Installation

How to install Birdhouse on your system.

## Prerequisites

- A modern operating system (macOS, Linux, or Windows)
- An API key for your preferred AI provider

## Install Birdhouse

Download the latest release from
[GitHub Releases](https://github.com/birdhouse-labs/birdhouse/releases).

## Next Steps

Once installed, head to the [Quick Start](/docs/getting-started/quick-start/)
guide to start using Birdhouse.
