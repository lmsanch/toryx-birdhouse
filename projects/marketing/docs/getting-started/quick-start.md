# Quick Start

Start using Birdhouse in minutes. This guide walks you through creating your
first agent team.

## Create a Workspace

Open Birdhouse and create a new workspace for your project.

## Start Your First Agent

Launch an agent and give it a task. The agent will have full access to your
workspace environment.

## Next Steps

Learn more about [Agent Teams](/docs/guides/agent-teams/) and how to collaborate
with multiple agents.
