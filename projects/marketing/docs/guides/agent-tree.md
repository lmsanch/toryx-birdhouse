# Agent Tree

Navigating and managing the agent tree in Birdhouse. The agent tree provides a
visual overview of all your agents and their relationships.

## Browsing Agents

Use the agent tree to browse active sessions and navigate directly to any agent.

## Team Structure

Agents are organized according to their team structure, making it easy to
understand the hierarchy.

## Activity Indicators

Visual highlighting shows which agents are actively working, so you can check on
progress at a glance.
